# Docker

## Ziel
Vyntrio Container Engine verwaltet Container, Stacks und App-Deployments über eine abstrahierte Benutzeroberfläche.

## Funktionen
- Images, Container, Volumes, Networks
- Compose Stacks
- Logs, Terminal, Restart, Update
- Backups und Templates

## Anforderungen
- Sichere Trennung zwischen System-Containern und User-Apps.
- Vorlagen mit validierten Eingaben.
- Backup- und Restore-Hooks für Apps.
- App Store Integration auf derselben Runtime.
